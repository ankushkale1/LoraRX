LoRa Receiver

Hardware:

1. ESP8266, preferably Node MCU v1
2. SX1278 LoRa module with antenna

PinOut:

![alt text](resources/image.png)

| NodeMCU Pins | SX1278 Pins |
| -------------- | ------------- |
| GND          | GND         |
| 3.3V         | VCC         |
| D8           | NSS         |
| D7           | MOSI        |
| D6           | MISO        |
| D5           | SCK         |
| D0           | RST         |
| D2           | DIO0        |
